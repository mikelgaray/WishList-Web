# WishList-Web

A simple wish list application web where you can add to a list, manage the lists and keep track of the games you want.

## What it does

This app lets users create and manage their personal video game wish lists. You can add games to a list want, modify those list by adding and deleting games. Built as a backend Php.

## Tech Stack

- Html
- Css
- JavaScript
- Php
- MySQL
